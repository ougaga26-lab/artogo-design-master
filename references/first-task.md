# 熱身任務選單 · First Task Menu

> 給 ARTOGO Design Master skill 在首次設定流程的 Step 5 使用。
> 目標：用一個 5 分鐘的小任務讓同事**親身體驗**設計工具的能力，建立信心。

---

## 三個熱身任務（讓同事 3 選 1）

### 選項 A · IG 貼文圖（1080×1080）

**腳本**：
> 「我們做一張 IG 貼文。給我一個主題就好——例如『下週公司聚餐』『新品上市』『感謝合作夥伴』。」

**等同事說主題後**，組 brief 給 huashu-design：

```
【任務】1080×1080 IG 貼文圖

【主題】[同事的主題]

【套用設計系統】
ARTOGO Design System
- 主色：#ba9972（金棕）
- 底色：#191919（深色）
- 字型：Roboto + Noto Sans TC
- 完整規範：~/.claude/skills/artogo-design-master/assets/brand-spec.md

【視覺要求】
- 單張獨立 HTML
- 中央視覺 + 邊框留白
- 至少一處「簽名細節」（大字序號 / 細水平線 / 全大寫眉題擇一）
- 反 AI slop 嚴格遵守
```

---

### 選項 B · 提案 PPT 封面（1920×1080）

**腳本**：
> 「我們做一張提案 PPT 封面。告訴我兩件事：
> 1. 主標題（例如「Q2 行銷規劃」「年度策略提案」）
> 2. 副標題或日期（例如「2026.05.10 · 內部會議」）」

**等同事回覆**，組 brief：

```
【任務】1920×1080 提案 PPT 封面（橫式）

【主標題】[同事提供]
【副標題 / 日期】[同事提供]

【套用設計系統】
ARTOGO Design System
（同上）

【視覺要求】
- 左主標 + 右側留白 / 細節元素的不對稱布局
- 主標 80-120px，金棕色或白色
- 底部 ARTOGO wordmark + 日期 / 編號
- 一個 hero 視覺（可以是純排版 / 一條金棕細線 / 序號）
```

---

### 選項 C · 公司活動海報（A4 直式 1240×1754）

**腳本**：
> 「我們做一張活動海報。告訴我四件事：
> 1. 活動名稱
> 2. 日期 + 時間
> 3. 地點
> 4. 一句活動 tagline（沒有也可以）」

**等同事回覆**，組 brief：

```
【任務】A4 直式海報 1240×1754

【活動內容】
- 名稱：[同事提供]
- 日期：[同事提供]
- 地點：[同事提供]
- Tagline：[同事提供 / 留空]

【套用設計系統】
ARTOGO Design System
（同上）

【視覺要求】
- 上 hero（活動名稱大字）+ 中段資訊條 + 下 ARTOGO logo
- 大字級節奏：主標 120-180px / 副標 32-48px / 內文 16-20px
- 右上角加序號或日期 monospace 標
- 底部要有 ARTOGO 細線 wordmark
```

## 如果同事說「我有別的想做」

不要堅持 A/B/C 三選一。讓同事自己描述需求，組 brief 直接交給 huashu-design 處理就好。記得 brief 永遠要附：

```
【套用設計系統】
ARTOGO Design System
完整規範：~/.claude/skills/artogo-design-master/assets/brand-spec.md
```

這個是 ARTOGO Design Master 跟普通 huashu-design 的核心差異——**永遠帶著公司規範入場**。
