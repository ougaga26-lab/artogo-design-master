嗨～ ARTOGO Design Master 是給公司同事用的設計助手。安裝完後，你可以在 Claude Code 裡直接說「ARTOGO 設計大師」，它就會自動啟用該 skill，並套用 ARTOGO 設計規範把成品交給你。下面是 5 步驟安裝指南，全程約 3 分鐘。

// 5 STEPS · ABOUT 3 MINUTES

怎麼開始

01

打開 Claude Code

如果還沒裝 Claude Code，先去 claude.ai/code 下載安裝（免費，跟著精靈一路下一步即可）。

02

複製這行指令貼到對話框

在 Claude Code 對話框裡貼下面這行，按 Enter。Claude 會自動下載 skill 到你電腦。

請執行：npx skills add ougaga26-lab/artogo-design-master -g --agent claude-code -y
⚠️ Claude 回「安裝完成」不是結束—— 這只是把檔案下載到你電腦，還沒設定。必須繼續做 Step 03-05，整個流程才算走完。
03

完全重啟 Claude Code

不是只關視窗——要在工作列右下角的 Claude Code 圖標 右鍵 → Quit / 結束，完全關掉它的背景程式，然後重新開啟。

為什麼要重啟？Claude Code 啟動時會掃描可用的 skill 清單，剛剛新裝的 skill 不重啟它讀不到——下一步打觸發詞就不會有反應。

⚠️ 這步省略 = Step 04 一定失敗。是 90% 同事卡關的原因
04

開新對話打這句話

在 Claude Code 開一個 全新對話（不是繼續用舊的），在對話框打下面這句話：

我要開始用 ARTOGO 設計工具

打了這句話之後，Claude 會自動接手做以下 4 件事，你只要看著就好：

①檢查你電腦的 Node.js 環境
②自動安裝設計引擎（huashu-design）
③寫入 ARTOGO 設計規範到設定檔
④給你一個熱身任務試手感

⏱ 全程約 2 分鐘。中間 Claude 如果問你問題，按提示回答即可。全程不用打複雜指令。

05

日常開始使用

設定完成後，之後每次想做設計，依序做這 3 個動作：

①

先指定本次的工作資料夾

在 Claude Code 對話框上方點資料夾名稱，切到你這次要存產出檔案的位置。

Local Design test ← 點這裡切換
Describe a task or ask a question...
// Claude Code 對話框
⚠️ 沒指定資料夾，生成的檔案會跑到奇怪的位置，等你要找的時候找不到
②

呼叫 ARTOGO 設計大師

在對話框打 下面任一句話 都會觸發：

ARTOGO Design Master ARTOGO 設計 ARTOGO 設計大師

送出後 Claude 會回覆，回覆訊息的最上方會出現這個標籤：

回覆訊息上方 › Ran skill /artogo-design-master ›
✓ 看到這個標籤 ＝ skill 真的被觸發了，安裝成功
③

開始請它做事

用日常語言描述你要的設計：

「幫我做下週活動的 IG 封面」
「做一份提案 PPT 第一頁」
「把這段內容做成 6 張多圖」

會自動套用 ARTOGO 金棕色 #ba9972 + 深色底 + Roboto/Noto Sans TC 字型。

遇到問題？

直接在 Claude Code 對話框打 「ARTOGO 跑不起來」 或 「重新設定 ARTOGO 設計工具」，Claude 會帶你疑難排解。還是搞不定？傳訊息給維護者（這個工具是 ARTOGO 內部專屬，不是商業產品，所以靠口耳相傳互相幫忙最快）。

CAPABILITIES · WHAT IT CAN DO

一句話 ＝ 一份能交付的設計

3 到 30 分鐘，你能 ship 一段產品發布動畫、一個能點擊的 App 原型、一套能編輯的 PPT、一份印刷級的信息圖。下面所有 GIF 都是花叔設計引擎跑出來的——不是 Figma，不是 AE，就是一句話 prompt。

CASE 01

iOS App 原型

iPhone 15 Pro 精確機身（靈動島 / 狀態列 / Home Indicator 像素級對齊）· 多屏狀態驅動切換 · 真實圖片從 Wikimedia / Met / Unsplash 自動取 · 交付前自動跑 Playwright 點擊測試。

// 怎麼觸發
「做個 AI 番茄鐘 iOS 原型，4 個核心螢幕要真能點擊」
CASE 02

Motion Design 動畫

時間片段模型 · useTime / useSprite / interpolate / Easing 四個 API 覆蓋所有動畫需求 · 一句話導出 MP4 / GIF / 60fps 插幀 / 帶 BGM 的成片，用於發布、宣傳、社群。

// 怎麼觸發
「把這段邏輯做成 60 秒動畫，導出 MP4 和 GIF」
CASE 03

HTML 幻燈片 → 可編輯 PPTX

HTML deck 在瀏覽器演講（鍵盤翻頁 / 全屏 / 計時器） · 同時導出 真實的 PowerPoint 文字框，PPT 裡雙擊就能編輯。簡報修改不用回到 Claude，直接在 PPT 改字。

// 怎麼觸發
「做一份 12 頁的 Q2 規劃 PPT，最後導出 PPTX 給我」
CASE 04

Tweaks 實時變體切換

配色 / 字型 / 信息密度等參數化，**側邊面板即時切換**對比效果。瀏覽器存 localStorage，刷新不丟。設計階段 review 利器，老闆喜歡哪個版本一鍵切換。

// 怎麼觸發
「給我這個 landing page 三個配色變體，可以切換」
CASE 05

信息圖 / 數據可視化

雜誌級排版 · CSS Grid 精準分欄 · 排印細節（text-wrap pretty / 全形引號 / letter-spacing 分層） · 真數據驅動 · 可導出 PDF 矢量 / PNG 300dpi / SVG。

// 怎麼觸發
「把這份營收數據做成一張 A4 直式信息圖」
CASE 06

5 維度專家評審

設計做完不確定好不好？讓 Master 從 5 個維度打分——哲學一致性 / 視覺層級 / 細節執行 / 功能性 / 創新性，各 0–10 分。輸出雷達圖 + Keep / Fix / Quick Wins 清單。

// 怎麼觸發
「幫我對這個設計做 5 維度評審，給我修復清單」
CASE 07

設計方向顧問

不知道要什麼風格？這個 Master 內建 5 流派 × 20 種設計哲學（信息建築 / 運動詩學 / 極簡 / 實驗先鋒 / 東方哲學）。它會推薦 3 個差異化方向，並行生成 3 個 Demo 讓你對比選擇。

// 怎麼觸發
「我要做品牌 landing page，推薦 3 個風格方向讓我選」
CASE 08

Junior Designer 工作流

Master 不會閉門造大招。**先寫 assumptions + placeholders + reasoning，盡早 show 給你**，等你點頭再迭代。理解錯了早改比晚改便宜 100 倍——這是 ARTOGO 工作流的核心原則。

// 自動觸發
每次設計任務都會自動套用，不需要特別講
CASE 09

品牌資產協議

做別家公司的設計（合作夥伴 / 客戶提案）時，Master 會自動執行 5 步資產協議：問 → 搜官網 → 下載 logo / 產品圖 / UI → 抽色值 → 寫 brand-spec.md。確保產出真的「像那個品牌」，不是套通用模板。

// 怎麼觸發
「做一份給 [客戶名] 的提案 PPT」（會自動觸發協議）
